# Fuu 🌿

**Breathe together with the world.**

世界中の人と同時に深呼吸するウェルネスアプリ。無料・登録不要・どこでも動く。

🔗 **Live: https://fukuokamotoshi-coder.github.io/breathsync/**

## Features

- 🫁 3 breathing patterns — 4-7-8, box breathing, relaxed
- 🌍 Real-time presence — see how many people are breathing with you right now
- 🎵 8 generative soundscapes — rain, ocean, forest, binaural beats & more (Web Audio API, no audio files)
- 👥 Friends & group challenges
- 🏠 Live themed rooms — morning routine, late night calm, focus
- 🔥 Streaks, badges & weekly stats
- 📱 PWA — installable, works offline

## Free forever

Every feature is free. The supporter plan (¥480/mo) exists for people who love the app and want to back it — it adds a supporter badge, extended history, custom breathing patterns, and early access to new features. It never locks core functionality.

## Stack

Single-file vanilla JS SPA · Web Audio API · Supabase Realtime (presence) · Stripe (supporter plan) · GitHub Pages
